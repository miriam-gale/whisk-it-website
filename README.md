# 🍰 Whisk It Cakes & Pastries Website

A modern and responsive bakery website designed for **Whisk It Cakes & Pastries**.  
This project showcases a clean user interface, smooth layout, and a visually appealing design built using HTML and CSS.

---

## ✨ Features

- 🎂 Beautiful hero section with call-to-action
- 🍩 Specialties section showcasing products
- 💕 About section with brand story and features
- 📞 Contact / CTA section for customer engagement
- 🌐 Fully responsive design (mobile, tablet, desktop)
- 🎨 Clean and modern UI design
